Proyecto

Los usuarios pueden agregar, modificar, eliminar y listar registros. Los registros se basan en tres valores clave: ID, DNI y Nombre. El ID es generado automáticamente por el sistema y no puede ser modificado. Todos los registros existentes se pueden visualizar en una tabla que muestra las columnas de ID, DNI y Nombre.

Datos:

  • ID
  • DNI
  • NOMBRES

Operaciones:

Agregar

Crea un nuevo registro. Se proporcionan DNI y Nombre, y el sistema genera un ID único automáticamente.

Modificar

Cambia los valores de DNI y Nombre de un registro existente, basándose en su ID. El ID no puede ser modificado.

Eliminar

Elimina un registro existente del sistema. Se debe proporcionar el ID del registro a eliminar.

Listar

Muestra todos los registros existentes en una tabla con las columnas ID, DNI y Nombre.
